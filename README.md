# 📝 Introduction

**Tic Tac Toe AI Game** is a desktop application developed using **C# and .NET WinForms**.  
The project recreates the classic Tic Tac Toe game with a clean interface and multiple gameplay modes.

The application focuses on implementing **game logic, AI decision-making, and interactive UI controls**.

# 🚀 Main Features

## 🎮 1. Multiple Game Modes
* **Player vs Player:** Two players can compete on the same device.
* **Player vs Computer:** Challenge the AI with different difficulty levels.

---

## 🧠 2. AI Difficulty Levels

The computer behavior changes depending on the selected level:

### Easy
* Random moves.

### Medium
* Defensive strategy.
* Attempts to block the player from winning.

### Hard
* Tries to win if possible.
* If not, blocks the opponent.
* Otherwise chooses a random move.

---

## 🧩 3. Game Logic System

* Board implemented using a **2D array**.
* Automatic detection of wins in:
  * Rows
  * Columns
  * Diagonals
* Real-time player turn tracking.
* Game state management (Win / Draw / In Progress).
